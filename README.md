**Линейка** — это простое и удобное расширение для браузера, которое добавляет в любое окно горизонтальные и вертикальные направляющие линии — точно так же, как в графических редакторах (Figma, Sketch, Photoshop, Adobe XD и др.).

Идеально подходит **верстальщикам**, **тестировщикам** и всем, кто занимается выравниванием элементов на веб-страницах.

## Для чего нужен PageLiner

- Быстро проверять выравнивание блоков, текстов, кнопок, изображений
- Сравнивать отступы между элементами
- Помогает при pixel-perfect верстке
- Удобно тестировать адаптивность и находить мелкие сдвиги
- Работает на **любом сайте** без необходимости вносить изменения в код

## Основные возможности

- Перетаскивание линий мышкой (горизонтальные + вертикальные)
- Сохранение позиций линий для конкретного сайта
- Изменение цвета линий
- Минимальное влияние на производительность
- Работает в Chrome, Chromium

## Как установить

1. Скачай архив
2. Открой Chrome → chrome://extensions/
3. Включи «Режим разработчика» (вверху справа)
4. Нажми «Загрузить распакованное расширение»
5. Выбери папку с проектом
Готово ✓

## Как пользоваться

1. Открой любую страницу
2. Нажми на иконку расширения
3. Выбирай действия в меню
4. !!! Желательно после использования отключать(на некоторых сайтах(1%) может ломать верстку)!!!
